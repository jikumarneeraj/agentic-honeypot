# Agentic Scam Honeypot

An autonomous AI agent that detects scam messages, engages scammers using a human-like persona, extracts intelligence such as UPI IDs and phishing links, and terminates conversations automatically.

## Features
- Scam detection
- Agentic conversation loop
- LLM-based questioning
- Intelligence extraction
- Session-based memory

## Run
```bash
pip install -r requirements.txt
uvicorn main:app --reload
